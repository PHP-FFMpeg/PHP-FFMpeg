addMetadata(array|null $data = null)

Add metadata to an audio file. If no arguments are given then filter will remove all metadata from the audio file

Parameters

array|null $data

If array must contain one of these key/value pairs:

  • "title": Title metadata
  • "artist": Artist metadata
  • "composer": Composer metadata
  • "album": Album metadata
  • "track": Track metadata
  • "artwork": Song artwork. String of file path
  • "year": Year metadata
  • "genre": Genre metadata
  • "description": Description metadata
